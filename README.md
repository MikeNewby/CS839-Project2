Project 2 - Implement a simple 3D dynamic elastic object

The project creates a cube of tetrahedra 

Compiled on Linux. from the P2 Folder:

$ cmake .

$ make

$ ./P2/P2

This will create NewbyP2.usda

$ usdview NewbyP2.usda
